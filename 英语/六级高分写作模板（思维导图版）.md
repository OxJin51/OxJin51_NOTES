# 六级高分写作模板（思维导图版）

## 核心必背：三大通用框架（骨架）

这部分内容**一字不差地背下来**，它们构成了你文章的80%。

### 1. 万能开头（二选一）

#### 场景A：名言/个人品质/抽象话题（如：自律、目标、信任）

> **背诵：** Amid today's multidimensional and escalating competition, success relies on capacities far exceeding technical competence. Therefore, the cultivation of **[主题词]** has gradually been recognized, yet its value remains underestimated, making it essential for us to strengthen it consciously. [cite: 2, 3]
>
> *中文：在当今多维且不断升级的竞争中，成功依赖的能力远超技术能力。因此，[主题词]的培养逐渐被认可，但其价值仍被低估，我们要有意识地加强它。*

#### 场景B：社会热点/现象类（如：AI、图书馆开放、网络暴力）

> **背诵：** In recent months, a discussion about **[现象/主题]** has repeatedly surfaced on social media trending topics. Yet beyond this visible trend lies a more essential question: what capacity must individuals strengthen to address such challenges wisely? [cite: 53, 54]
>
> *中文：最近关于[主题]的讨论频上热搜。但在趋势背后有个核心问题：我们要加强什么能力来明智应对？*

---

### 2. 万能中间段（论证逻辑链）

不要去想具体的理由，直接套用这套**"心态转变 -> 摒弃错误 -> 采取行动 -> 升华"**的逻辑。

#### 中间段四步结构：

1. **心态转变**
   > To act on this insight, the first shift must happen at the level of mindset. [cite: 84]

2. **摒弃旧习**
   > Specifically, whether in increasingly competitive contexts or fast-paced workplaces, everyone should decisively abandon the belief that **[错误观念/被动等待]**. [cite: 74, 76]

3. **正确行动**
   > Instead, a more responsible approach is to **[正确做法/主动探索]**. Promoting this provides us with a structured channel to develop **[核心能力]**, preparing us more effectively for future challenges. [cite: 69, 92]

4. **升华意义**
   > Ultimately, it is this process—not the action itself—that enables people to internalize deeper qualities and apply them in complex, real-world environments. [cite: 93]

---

### 3. 万能结尾（强力收束）

> **背诵：** Now, can we afford to move forward without **[主题词]**? As adaptability increasingly eclipses stagnation, **[主题词]** has shifted from optional to essential for lasting success. It demands building new inner strengths so individuals can influence what comes next rather than merely withstand it. [cite: 94, 105, 107]

## 关键填空：三组话题"补丁包"（血肉）

考试时，把下面这些词块填进上面框架的 **[括号]** 里。

### 1. 科技与未来类（AI、数字化、信息过载）

| 类别         | 英文表达                                    | 中文含义                  |
| ------------ | ------------------------------------------- | ------------------------- |
| **核心能力** | `digital literacy`                          | 数字素养                  |
|              | `independent thinking`                      | 独立思考 [cite: 70, 122]  |
| **错误观念** | `waiting for others to teach`               | 等别人教 [cite: 77]       |
|              | `forwarding posts without checking sources` | 不查来源就转发 [cite: 58] |
| **正确做法** | `verify information`                        | 核实信息                  |
|              | `use tools appropriately`                   | 适当地使用工具            |

### 2. 个人成长与职场类（自律、坚持、逆商）

| 类别         | 英文表达                     | 中文含义                     |
| ------------ | ---------------------------- | ---------------------------- |
| **核心能力** | `resilience`                 | 韧性 [cite: 84]              |
|              | `self-discipline`            | 自律                         |
|              | `interdisciplinary thinking` | 跨学科思维 [cite: 86]        |
| **错误观念** | `setting unrealistic goals`  | 设立不切实际的目标 [cite: 7] |
|              | `dismissing it as trivial`   | 认为它不重要 [cite: 5]       |
| **正确做法** | `set realistic goals`        | 设实际目标 [cite: 6]         |
|              | `work persistently`          | 坚持不懈                     |

### 3. 传统与社会类（老年人、传统文化、代沟）

| 类别         | 英文表达                | 中文含义            |
| ------------ | ----------------------- | ------------------- |
| **核心能力** | `cultural awareness`    | 文化意识 [cite: 71] |
|              | `rationality`           | 理性 [cite: 122]    |
| **特制补丁** | 遇到老年人/传统话题必用 |                     |

> **特制补丁句：** Dismissing these "old-fashioned" ideas as useless is itself a cognitive bias. The very reason they have endured is that they remain resilient over time, retaining practical value. [cite: 116, 117]
>
> *把这一句插在中间段，替换掉"错误观念"那一句*

## 考场极速操作指南

### 操作流程

1. **看题定类型**
   - 是"个人/名言" → 用**开头A**
   - 是"社会现象" → 用**开头B**

2. **默写框架**
   - 先在草稿纸或脑海中把三大框架（开头、中间、结尾）搭好

3. **填入词块**
   - **AI题**：填入 `digital literacy` 和 `verify information`
   - **传统文化题**：填入 `cultural awareness` 并用上"认知偏差"补丁句
   - **普通励志题**：填入 `resilience` 和 `work persistently`

### 极限备考任务（Next Step）

> **立即行动：** 请现在尝试只看中文，口头复述一遍**开头A**和**万能结尾**的英文。这两个是最容易拿分且最通用的部分。
