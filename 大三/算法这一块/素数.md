# 素数

version1

```cpp
int countPrimes(int n) 
{
    int count = 0;
    for (int i = 2; i < n; i++)
        if (isPrime(i)) count++;
    return count;
}
// 判断整数 n 是否是素数
bool isPrime(int n) 
{
    for (int i = 2; i < n; i++)
        if (n % i == 0)
            // 有其他整除因子
            return false;
    return true;
}
```

```cpp
class Solution {
public:
    int countPrimes(int n) 
    {
        vector<bool> isPrime(n, true);
        for (int i = 2; i * i < n; i++) 
        {
            if (isPrime[i]) 
            {
                for (int j = i * i; j < n; j += i) 
                {
                    isPrime[j] = false;
                }
            }
        }
        
        int count = 0;
        for (int i = 2; i < n; i++) 
        {
            if (isPrime[i]) 
            {
                count++;
            }
        }
        
        return count;
    }
};
```



### **题目描述**

给定一个正整数 `n`(n > 1)，找出所有小于等于 `n`的质数，然后计算这些质数中其十进制表示包含数字 “2” 的所有质数的乘积，并输出最终结果。

**输入格式：**

一个正整数 `n`

**输出格式：**

一个整数，表示所有小于等于 `n`的包含数字 “2” 的质数的乘积。

**输入样例 1：**

```
5
```

**输出样例 1：**

```
2
```

**解释：**

小于等于5的质数有：2, 3, 5。其中包含数字“2”的质数只有它本身（2）。因此，乘积就是 2。

**输入样例 2：**

```
1000
```

**输出样例 2：**

```
1593228429288253996762335432319716869455928388181527321278398654529458
```
